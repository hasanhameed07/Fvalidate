FvalidateA web2.0 form validation plugin for jQuery.

Inline Rules

Javascript

$("#myform1").Fvalidate();

Confirm Field


Userid doesn't match

Javascript

$("#myform2").Fvalidate(); // this will attach the confirm validation to both the fields $("input[name=confirm-userId]").sameAs($("input[name=userId]"));

Length Check

Javascript

$("#myform3").Fvalidate();

Regex Rules

Javascript

$("#myform5").Fvalidate(); // define your custom regex rules to the fields $("input[name=address]").rule(/^[a-z0-9-\s]*$/); $("input[name=city]").rule(/^[a-zA-Z\.\s]*$/);

Attach to Events


Submit with Alert | Set Error | Remove Error | Count Errors

Javascript

$("#myform6").Fvalidate({ submitButton: $("#confirm"), alertErrors: true, onSuccess: function(){ alert('Success'); return false; // stop form submission }, onError: function(errors){ console.log(errors); } }); $("a#setError").click(function(){ $("input[name=event-username]").setError('Hmm.. A Custom Error!'); return false; }); $("a#removeError").click(function(){ $("input[name=event-username]").removeError(); return false; }); $("a#countError").click(function(){ var count = $("#myform6").countErrors(); alert(count); return false; });
